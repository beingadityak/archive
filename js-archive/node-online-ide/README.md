# Online Language IDE

In this project, I created a simple IDE in NodeJS and used a lesser-known module called **CompileX**.
This project runs on Windows systems only.

The languages it can compile are:

  1. C/C++ (gcc MinGW)
  2. Java (javac)
  3. VB.NET (vbc)
  4. Python (python27)
  5. C# (csc)
